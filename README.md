# shieldz-blog

The [Shieldz](https://shieldz.cash) blog, built with [Astro](https://astro.build).
Served at https://shieldz.cash/blog.

```bash
npm install
npm run dev      # local
npm run build    # static -> dist/
scripts/deploy.sh  # build + rsync to prod (SSH key)
```

Posts live in `src/content/blog/*.md`.
